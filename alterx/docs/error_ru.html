Главное меню(en)
Сообщения об ошибках:
Срабатывание концевых выключателей (joint X limit error)

Наиболее распространенной причиной срабатывания концевых выключателей является:
1. Ослабление муфты, связывающей двигатель с осью или с датчиком
2. Наводки на цифровой вход концевого выключателя.
3. Неверная настройка программных ограничений позиции.

Неравномерное движение оси

Наиболее распространенной причиной неравномерного движения оси является:
1. Ослабление муфты, связывающей датчик с осью.
2. Пульсации обратной связи двигателя.
3. Неверная настройка PID-регулятора оси.
4. Механическая проблема с осью.
5. Пропуск сигналов обратной связи по причине повреждения датчика.

Ошибка рассогласования (joint X following error)

Наиболее распространенной причиной появления ошибки рассогласования оси является:
1. Неверная настройка PID-регулятора оси.
2. Неверная настройка электропривода оси.
3. Отсутствие или пропуски сигналов от датчиков обратной связи.
4. Механическая проблема с осью.

Ошибка чтения (error finishing read)

Наиболее распространенной причиной появления ошибки чтения является:
1. Низкокачественный или поврежденный кабель связи блока вычисления с контроллером автоматики.
2. Неверная настройка сетевой карты блока вычисления или проблема с драйвером.

Нет запуска интерфейса

Ошибки запуска интерфейса связаны с:
1. Неверно настроенная конфигурация СЧПУ (*.ini и *.hal файлы).
2. Нет связи с контроллером автоматики (не обязательно для симуляции).

Нет запуска операционной системы

Прерывание загрузки операционной системы свидетельствует о неисправности компонентов системы или другой ошибке.
Данная проблема решается исключительно в сервисном центре.

Ошибка задержки реального времени (Unexpected realtime delay)

Ошибка задержки реального времени не является критической и означает,
что разово превышен лимит времени между запросами данных.

Наиболее вероятной причиной является:
1. Неверная настройка процессора блока вычисления.
2. Перегрузка процессора второстепенными задачами.

Ошибка связи (Communication error)

Наиболее вероятной причиной ошибки связи является:
1. Помехи на шине расширения блока автоматики.
2. Неправильно собранный корпус блока автоматики.

Используется неизвестный G-код (Unknown g code used)

Необходимо проверить правильность написания технологической программы.
При использовании пользовательских М-кодов необходимо убедиться,
что файлы подпрограммы находятся в директории заданной параметрами.

Выход за предел оси (Linear move on line N would exceed Axis N’s negative/positive limit)

Наиболее распространенной причиной движения оси за пределы программного лимита является:
1.Неправильная технологическая программа.
2. Неверная настройка программных ограничений позиции.

Не могу выполнить действие (can’t do that (X) in auto mode with interpreter waiting)

Планировщик заданий занят и не может выполнить действие Х.
Ошибка может возникать, например, в случае запуска программы в момент когда СЧПУ уже работает по программе.

Неизвестный символ (Bad charaster “X”used)

Наиболее распространенной причиной такой ошибки является:
1. В программе обнаружен неиспользуемый в RS274 символ.
2. Пропущена буква в слове кадра.
3. При написании комментария не указали символ начала комментария.
